# math.atan2

2つの引数のアークタンジェントを返す

```lua
math.atan2 (y, x)
```

## 説明

`y / x` のアークタンジェント（逆正接）をラジアン単位で返します。`math.atan2` は符号を考慮して結果の象限を正確に決定します。

## サンプルコード

```lua
local y, x = 1, 1
print(math.atan2(y, x))  -- ラジアン単位での結果
```

この例では、`y / x`のアークタンジェントを計算します。

## 互換性

- Lua 5.1
