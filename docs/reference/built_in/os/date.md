# os.date

```lua
os.date ([format [, time]])
```

## 説明

指定された文字列フォーマットに従って、日付と時刻をフォーマットして返します。フォーマットが `'!*t'` の場合は、UTC時刻を返します。フォーマットに`'*t'`を指定すると、年、月、日、時間などのフィールドを持つテーブルを返します。

## サンプルコード

```lua
print(os.date("%Y-%m-%d %H:%M:%S"))  -- 現在の日付と時刻を表示
```

この例では、`os.date`を使って、現在の日時をフォーマットしています。

## 互換性

- Lua 5.1
