# tostring

値を文字列に変換する

```lua
tostring (e)
```

## 説明

任意の型の引数 `e` を受け取り、それを適切な文字列に変換します。数値の変換を細かく制御したい場合は、`string.format` を使用してください。

`e` のメタテーブルに `"__tostring"` フィールドがある場合、`tostring` はそのフィールドに関連付けられた関数を `e` を引数として呼び出し、その結果を返します。

## サンプルコード

```lua
print(tostring(123))  -- "123"
```

この例では、数値が文字列に変換されます。

## LuaJIT独自の拡張

NaNと±Infを正規化します。

## 互換性

- Lua5.1
- LuaJIT独自の拡張あり
