# lua_getfenv

指定されたインデックスにある値の環境テーブルをスタックにプッシュします。

`[-0, +1, -]`

```c
void lua_getfenv (lua_State *L, int index);
```

## 説明

指定されたインデックスにある値の環境テーブルをスタックにプッシュします。

## サンプルコード

```c
lua_getfenv(L, 1);
```

このコードは、インデックス1の環境テーブルを取得し、スタックにプッシュします。

## 互換性

- Lua5.1
