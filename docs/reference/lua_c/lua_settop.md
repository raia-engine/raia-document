# lua_settop

スタックのトップを指定されたインデックスに設定します。

`[-?, +?, -]`

```c
void lua_settop (lua_State *L, int index);
```

## 説明

任意の有効なインデックス、または0を受け付け、スタックのトップをこのインデックスに設定します。新しいトップが古いものより大きい場合、新しい要素はnilで埋められます。インデックスが0の場合、すべてのスタック要素が削除されます。

## サンプルコード

```c
lua_settop(L, 0);
```

このコードは、スタックのすべての要素を削除します。

## 互換性

- Lua5.1
