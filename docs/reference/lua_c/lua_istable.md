# lua_istable

指定されたインデックスにある値がテーブルであれば1を返します。

`[-0, +0, -]`

```c
int lua_istable (lua_State *L, int index);
```

## 説明

指定された許容インデックスにある値がテーブルであれば1を返し、そうでなければ0を返します。

## サンプルコード

```c
if (lua_istable(L, 1)) {
    printf("値はテーブルです\n");
}
```

このコードは、インデックス1の値がテーブルであるかを確認し、テーブルの場合にメッセージを表示します。

## 互換性

- Lua5.1
